Nash equilibrium


A Nash equilibrium of a game is a set of (possibly mixed) strategies σ=(σ1,…,σn) such that, if each player i believes that that every other player j will play σj, then i should play σi. That is, when ui is the utility function for the i-th player:

σi≠σi′→ui⁢(σi,σ-1)>ui⁢(σi′,σ-1)
∀i≤n⁢ and ⁢∀σi′∈Σi

Translated, this says that if any player plays any strategy other than the one in the Nash equilibrium then that player would do worse than playing the Nash equilibrium.
